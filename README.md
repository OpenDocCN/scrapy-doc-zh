## 下载

### Docker

```
docker pull apachecn0/scrapy-doc-zh
docker run -tid -p <port>:80 apachecn0/scrapy-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g scrapy-doc-zh
scrapy-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
